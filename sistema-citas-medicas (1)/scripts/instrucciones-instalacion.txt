INSTRUCCIONES DE INSTALACIÓN - Sistema de Gestión de Citas Médicas
================================================================

REQUISITOS PREVIOS:
- Node.js versión 14 o superior
- MySQL versión 8.0 o superior
- Editor de código (VS Code recomendado)

PASOS DE INSTALACIÓN:

1. PREPARAR EL ENTORNO
   - Descargar e instalar Node.js desde https://nodejs.org
   - Descargar e instalar MySQL desde https://dev.mysql.com/downloads/
   - Crear una carpeta para el proyecto

2. CONFIGURAR LA BASE DE DATOS
   - Abrir MySQL Workbench o línea de comandos de MySQL
   - Ejecutar el archivo: scripts/01-create-database.sql
   - Ejecutar el archivo: scripts/02-insert-test-data.sql
   - Verificar que las tablas se crearon correctamente

3. CONFIGURAR EL PROYECTO
   - Copiar el archivo .env.example y renombrarlo a .env
   - Editar el archivo .env con tus datos de MySQL:
     * DB_HOST=localhost
     * DB_USER=tu_usuario_mysql
     * DB_PASSWORD=tu_password_mysql
     * DB_NAME=sistema_citas_medicas
     * JWT_SECRET=una_clave_secreta_muy_larga_y_segura
     * PORT=3000

4. INSTALAR DEPENDENCIAS
   - Abrir terminal en la carpeta del proyecto
   - Ejecutar: npm install
   - Esperar a que se instalen todas las dependencias

5. INICIAR EL SERVIDOR
   - En la terminal, ejecutar: npm start
   - El servidor se iniciará en http://localhost:3000
   - Verificar que aparezca el mensaje de conexión exitosa

6. PROBAR LA APLICACIÓN
   - Abrir navegador web
   - Ir a http://localhost:3000
   - Probar con los usuarios de prueba incluidos

SOLUCIÓN DE PROBLEMAS COMUNES:
==============================

1. Error de conexión a MySQL:
   - Verificar que MySQL esté ejecutándose
   - Comprobar credenciales en el archivo .env
   - Verificar que la base de datos existe

2. Error "Cannot find module":
   - Ejecutar: npm install
   - Verificar que Node.js esté instalado correctamente

3. Puerto ocupado:
   - Cambiar el puerto en el archivo .env
   - O cerrar la aplicación que esté usando el puerto 3000

4. Problemas con JWT:
   - Verificar que JWT_SECRET esté configurado en .env
   - La clave debe ser larga y segura
